# 访谈录音转写整理器（Interview Transcript Organizer）

[![中文](https://img.shields.io/badge/lang-中文-red.svg)](README.md)
[![English](https://img.shields.io/badge/lang-English-blue.svg)](README.en.md)

一个 Claude Code Skill，将访谈录音转写文本自动整理为格式规范、结构清晰的 Word 文档。

## 简介

做访谈最痛苦的不是录音，而是整理成稿。 raw transcript 里塞满了「呃」「嗯」「就是就是」，大段回答没有段落，时间线也乱成一团。

这个 Skill 帮你把机械性排版工作全包了：
- 按主题自动分章
- 梳理 Q&A，记者问题蓝色加粗，受访者回答黑色左对齐
- 语义分段，把一大坨回答按意思转折拆开
- 清洗口头禅和语音冗余
- 校正语音识别错误，统一专业术语和人名
- 最终输出严格符合排版要求的 `.docx` Word 文档

## 功能特点

| 功能 | 说明 |
|------|------|
| **主题分章** | 按内容逻辑切分为 4~8 个主题章节，使用中文大写编号 |
| **Q&A 梳理** | 记者与受访者身份明确，问题/回答层次分明 |
| **语义分段** | 严禁堆成一整段，按意思转折拆成自然段 |
| **口语清洗** | 删除填充词、断线确认、口头禅，保留信息量 |
| **转写校正** | 修正语音识别错误，补充术语全称，统一命名 |
| **格式规范** | 楷体+Times New Roman，章节标题灰色底纹，严格排版 |

## 触发方式

在 Claude Code 中安装本 Skill 后，直接说：

- "帮我整理这个访谈录音"
- "把这个访谈录音生成 Word 文档"
- "访谈录音整理"
- "录音整理成 Word"
- "梳理问题和回答"

## 格式规范速览

- **文档格式**：`.docx`，A4 竖版
- **字体**：汉字楷体（KaiTi），英文/数字 Times New Roman，正文 10.5 pt
- **主标题**：16 pt 加粗居中
- **章节标题**：14 pt 加粗，带灰色文字底纹（RGB `D9D9D9`）
- **记者问题**：蓝色加粗（RGB `0, 112, 192`），前缀 `【记者】`
- **受访者回答**：黑色不加粗，前缀 `【受访者】`，左对齐无缩进

> 详细排版规则请见 `SKILL.md`

## 安装方式

1. 将本文件夹复制到你的 Claude Code skills 目录：
   ```bash
   cp -r interview-transcript-organizer ~/.claude/skills/
   ```
2. 重启 Claude Code 或重新加载 skills。
3. 直接说出触发词即可使用。

## 使用示例

**用户输入：**
> 帮我整理这个访谈录音，生成 Word 文档。这是转写文本：[粘贴文本]

**Skill 处理：**
1. 提取受访者姓名、采访者、(需要对访谈者进行匿名处理，例如首字母缩写或者昵称)、访谈形式
2. 按主题切分章节
3. 清洗冗余、校正错误、语义分段
4. 应用格式并生成 `.docx`
5. 询问保存路径后输出文件
